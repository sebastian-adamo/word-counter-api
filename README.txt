Install Instructions: 
	- Import root folder into preferred IDE and run 'WordCounterApplication" main method.
	- The server will start on 'http://localhost:8080'.

Note: 
	Views have been used for simplicity and demonstration purposes, however methods can easily be changed to return JSON data for example.

Assumptions:
	- Text body is grammatically correct.
	- Word length does not include punctuation.
	- White spaces and new lines have been skipped.
	- '/' characters have been counted as part of a word as they are not part of correct english punctuation and are generally used for number or date formatting.
	- '&' has been counted as a word.	
	- Average word length has been rounded to 3 decimal places.
